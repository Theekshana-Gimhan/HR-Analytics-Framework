# Reference [6]: Serverless Economics (N. Kodakandla, 2021)

### **Core Findings**
Serverless architectures provide significant cost and scalability advantages for cloud-native applications.

### **Why Serverless is Best for Your Project**
1.  **Granular Billing:** You pay for execution time, not idle time.
2.  **Scale-to-Zero:** Costs are literally zero when the system is not in use (e.g., at night).
3.  **Reduced TCO:** Total Cost of Ownership is lower because the cloud provider handles server maintenance and scaling.
4.  **Optimal for Batch:** Perfect for your nightly "Batch Prediction" job.
