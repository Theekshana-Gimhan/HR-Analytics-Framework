# Reference [5]: Google Vertex AI Features

### **Core Features for Your Project**
*   **AutoML Tabular:** Automatically creates, trains, and tunes the best model for structured data (like your HR records).
*   **Batch Prediction:** Allows you to process large amounts of data asynchronously. This is the "Serverless" heart of your system.
*   **Explainable AI (XAI):** Provides **Feature Attributions** (SHAP values) that explain *why* a certain prediction was made.
*   **BigQuery Integration:** Simplifies data storage and feature extraction.

### **Use in Viva**
"I am using Google Vertex AI's **Batch Prediction** feature. This allows the system to remain cost-effective by only using compute resources during the nightly prediction run, rather than maintaining an expensive always-on endpoint."
