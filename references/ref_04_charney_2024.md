# Reference [4]: HR Software Pricing Guide (B. Charney, 2024)

### **Core Findings**
International enterprise HR software uses a **Per-Employee-Per-Month (PEPM)** pricing model.

### **Typical Costs**
*   **Core HR:** $5 – $20 PEPM (~LKR 1,500 – 6,000).
*   **Full Suite (Payroll + AI):** $15 – $30+ PEPM (~LKR 4,500 – 9,000+).

### **Use in Viva (The "Killer" Statistic)**
"While international platforms cost an average of **LKR 2,400 to 9,000 per employee**, my system targets a total cost of **LKR 200 per employee**. This represents a 90%+ cost reduction, which is the primary driver for SME adoption in Sri Lanka."
